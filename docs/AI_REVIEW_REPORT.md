## 3. 解消済みの前回懸念

### 3.1 追加指標の交渉ポイント反映

前回は、分野の `relatedIndicatorIds` には入っているが、交渉ポイントの `indicatorIds` には出てこない指標があった。現在は以下が反映済み。

| 分野 | 対応 |
|---|---|
| 研磨 | `sppi-industrial-waste-disposal` を研磨材・処理費の説明ポイントへ追加 |
| 板金 | `cgpi-bolts-nuts` 用の購入部品ポイントを追加 |
| めっき | `cgpi-water` と `sppi-industrial-waste-disposal` を用水・処理費の説明へ追加 |
| 熱処理 | `cgpi-lpg` と `cgpi-heavy-oil-a` を炉エネルギー説明へ追加 |

なお、分野指標のうち一部は補助的な指標として交渉ポイント未使用のまま残っているが、分野の説明資料として自然な範囲。

### 3.2 ドキュメント表記

`docs/DATA_UPDATE.md` の指標数は「28指標（CGPI 26 + SPPI 1 + 最低賃金 1）」に更新済み。

### 3.3 分野ページの出典概要

分野ページ下部の「出典概要」は、`SourceSummary.label` と `seriesCode` を使い、各指標のラベルと接頭辞付きデータコードを表示するようになった。`/sources`、指標カード詳細、印刷シートでも系列追跡性は維持されている。

## 4. 残る注意点

大きなデータ整合・表示整合の懸念はなし。

次に触る UI リデザインでは、以下のガードレールを維持すること。

- 印刷シートは白背景・黒文字のビジネス文書調を維持する。
- 公的データのグラフ・数値・出典リンク・データコード・原データ CSV リンクを削らない。
- 取適法の法令文言、条文番号、免責、参考注記を意味変更しない。
- SNS 向けに見た目を柔らかくしても、「価格改定を煽る」表現にはしない。
